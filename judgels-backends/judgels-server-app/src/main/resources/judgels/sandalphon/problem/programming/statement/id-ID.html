Deskripsi

INI ADALAH CONTOH SOAL.

Anda memiliki $N$ kotak. Kotak ke-$i$ berisi $A_i$ apel.

Tentukan jumlah apel dari seluruh kotak tersebut!

Batasan

  • $1 \leq N \leq 1000$
  • $1 \leq A_i \leq 1\,000\,000$

Subsoal

  1. (50 poin) $A_i \leq 1000$
  2. (50 poin) Tidak ada batasan tambahan.

Masukan

Masukan diberikan dalam format berikut:

N
A1 A2 … AN

Keluaran

Keluarkan sebuah baris berisi jumlah apel dari seluruh kotak.

Contoh Masukan 1

2
3 4

Contoh Keluaran 1

7

Penjelasan Contoh 1

Anda memiliki $2$ kotak, yang berisi $3$ dan $4$ apel. Jumlah apel dari kedua kotak tersebut adalah $3 + 4 = 7$.

Contoh Masukan 2

1
42

Contoh Keluaran 2

42
