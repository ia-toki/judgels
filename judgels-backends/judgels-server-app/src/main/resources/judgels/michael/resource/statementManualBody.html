Formatting manual

Inserting PDF

  • Upload the PDF as a media file.
  • Switch to Source mode.
  • Add it as <embed src="render/<filename>" type="application/pdf"></embed>.

Inserting image

  • Upload the image as a media file.
  • Switch to Source mode.
  • Add it as render/<filename>.

Inserting spoiler

  • Switch to Source mode.
  • Add it as:
  • <div class="spoiler">Spoiler title
    <div class="spoiler-content">
        Spoiler content
    </div>
</div>
